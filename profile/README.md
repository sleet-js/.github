# SLEET JS
js playground by sleet to help in the development of near dapps.

---

When it comes to makeing near dapps for the web you have two options, near api js with like 10 plus diffent pacages for @near-js this or @near-js that and that is not counding wallet connection dependencies.

Or you could use one package. fastintear.

both options have their use cases and limitaions.

- near api js :: https://docs.near.org/tools/near-api
- fastintear :: https://github.com/elliotBraem/fastintear
other near js options
- near-kit :: https://github.com/r-near/near-kit.git
- near-jsonrpc-client-ts :: https://github.com/near/near-jsonrpc-client-ts
