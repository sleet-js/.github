# SLEET JS
js playground by sleet to help in the development of near dapps. (it's mostly ts)

---

### NEAR JS

When it comes to makeing near dapps for the web you have a few js options

- near api js :: https://github.com/near/near-api-js
- fastintear :: https://github.com/elliotBraem/fastintear
- near-kit :: https://github.com/r-near/near-kit.git
- near-jsonrpc-client-ts :: https://github.com/near/near-jsonrpc-client-ts
- near-connect (wallet connection) :: https://github.com/azbang/near-connect

See more at https://docs.near.org/tools/near-api
